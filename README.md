## Getting Started

1. `$ git clone git@github.com:aomra015/curry.git`
2. `$ cd curry`
3. `$ bundle install`
4. `$ rake db:migrate`

### Start development environment (at localhost:3000)
- `$ foreman start`

### Run tests
- `$ rake test`
